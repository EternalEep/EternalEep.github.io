Leetcode387 - First Unique Character in a String

Posted on 2018-09-25 | Edited on 2018-10-28

Solution

  • Approach 1
    • using (cnt[p-‘a’]) to record the alphabet
    • count (cnt[s[i]-‘a’]==1) then return i
    • with s.size(), not cnt.size()
  • Complexity Analysis
    • Time complexity: O(n)
    • Space complexity: O(1)
  • Approach 2
    • traverse the s
  • Complexity Analysis
    • Time complexity: O(n^2)
    • Space complexity: O(1)
