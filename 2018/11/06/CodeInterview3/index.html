CodeInterview3-数组中重复的数字

Posted on 2018-11-06 | Edited on 2018-11-15

Solution

  • Approach
    • Loop nums[i]!=i very important! Loop until nums with index i exists nums[i].
    • If yes, return true
    • If no, swap(nums[i],nums[nums[i]])
    • Pay attention to *duplication usage!
  • Complexity Analysis
    • Time complexity: $O(n)$
    • Space complexity: $O(1)$
    • Runtime: 3ms
