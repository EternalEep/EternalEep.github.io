Leetcode217-Contains Duplicate

Posted on 2018-11-02 | Edited on 2018-11-15

Solution

  • The number range is not in [1,n], Attention!
  • Related Topics
    • Leetcode287-Find the Duplicate Number
    • CodeInterview3-数组中重复的数字

Approach 1 - Hash Map

  • Hash Map
    • For self-balancing Binary Search Tree, search and insert are both $O(logn)$
    • For Hash Map, search and insert are both $O(1)$
  • Complexity Analysis
    • Time complexity: O(n)
    • Space complexity: O(n)
    • Runtime: 24ms

Approach 2 - Sorting

  • Sorting
    • If there are any duplicate integers, they will be consecutive after sorting.
  • Complexity Analysis
    • Time complexity: $O(nlogn)$ for heapsort
    • Space complexity: $O(1)$ for heapsort
    • Runtime: 28ms

Note

  • One should keep in mind that real world performance can be different from what the Big-O notation says. The Big-O notation only tells us that for sufficiently large input, one will be faster than the other.
  • Therefore, when nn is not sufficiently large, an $O(n)$ algorithm can be slower than an $O(nlogn)$ algorithm.
