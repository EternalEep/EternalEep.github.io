Leetcode1-Two Sum

Posted on 2018-10-27 | Edited on 2018-10-28

Solution 1

  • Approach
    • Brute Force
      • Loop through each element and find x if there is another value that equals to target-x
  • Complexity Analysis
    • Time complexity: O(n^2)
    • Space complexity: O(1)
    • Runtime: 172ms/92ms

Solution 2

  • Approach
    • One-pass Hash Table
      • While we iterate and inserting elements into the table, we also look back to check if current element’s complement already exsits in the table. If it exists, we have found a solution and return immediately.
  • Complexity Analysis
    • Time complexity: O(n)
    • Space complexity: O(1)
    • Runtime: 8ms
