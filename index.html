Leetcode189-Rotate Array

Posted on 2018-09-01

该题的整个多种方法解题思路见链接

Submission 3

  • Using Extra Array
    • 该方法使用了额外的空间，我只存储了数组的前k个数字，剩下的直接右移动k位以后，再将前k个移到数组中的相应位置，至于为什么要先额外存储前k个，因为我们知道，这样将不会丢失数组的原始元素。

Submission 4

  • Brute Force
    • 提交失败，最终结果为Time Limit Exceeded。
    • 外循环k次，每次数组右移1次

Submission 5

  • Using Reverse
    • 该方法是用了reverse法，即将数组先倒序，再将前k个与后len-k个倒序既能得到最终答案。这是一个非常巧妙的办法。
      *需要注意的是，reverse函数中的while循环条件，应选择”start<\end”，而不是”start=end”。”start=end”时，数组奇数个元素的时候没问题，偶数个元素的时候，会跳过”start=end”这一步，直接是start+1>end-1。

Submission 6

  • Using Cyclic Replacements
    • 该思路也是一个trick
    • 每隔k个进行循环替换，这样避免了元素消失问题，也不需要额外的空间
    • 每次循环次数为循环替换到最初位置的时候
    • 总循环次数为${n/k * k = n}$次，跳出外循环

Leetcode122-Best Time to Buy and Sell Stock II

Posted on 2018-08-25 | Edited on 2018-09-01

Submission 1

  • 如果第二个数是大于前一个数的，将其加入profit，从头到尾遍历该数组，最后所得profit即为我们所需要的最大利润
    • 例如[1, 7, 2, 3, 6, 7, 6, 7]
      • (3-2)+(6-3)+(7-6)=7-2=5

Submission 2

  • 寻找连续的peak及value值，需要注意的几个点有：
    1. i从0开始，并且小于数组长度-1
    2. prices[i]>=prices[i+1]
    3. prices[i]<=prices[i+1]
    4. 需要注意等号,否则测试用例{3,3}，i不增长，一直跳不出外层while循环

Leetcode26-Remove Duplicates from Sorted Array

Posted on 2018-08-20 | Edited on 2018-09-01

Submission 3

  • 使用unique函数：去重函数，去除相邻重复元素(保留一个)，把重复的元素后移动了，依然保存在原数组中，返回去重后最后一个元素的地址。由于去除相邻重复元素，一般用之前需要排序。

Submission 5

  • 先判断长度是否为0比后判断，速度更快
  • 因为最终只要返回不重复元素的长度，程序会自动返回前n个长度的数组，直接统计不重复元素个数，并且将不重复元素前移，注意审题。
